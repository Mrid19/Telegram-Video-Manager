## How to Install (Developer Mode)
Since this tool is not in the Chrome Web Store, you must install it manually:

1. **Download this Code:**
   - Click the green **Code** button above -> **Download ZIP**.
   - Unzip the file to a folder on your desktop.

2. **Load into Chrome/Brave:**
   - Open your browser and go to `chrome://extensions` (or `brave://extensions`).
   - Turn on **Developer Mode** (top right switch).
   - Click **Load Unpacked**.
   - Select the folder you just unzipped.

3. **Usage:**
   - Go to [Telegram Web](https://web.telegram.org/).
   - Play any video.
   - Click the extension icon to see the download button.
